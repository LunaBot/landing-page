I've spent the last few months using Discord servers. Big and small, they all had the same problem. A lack of a good all-in-one bot.

I want to fix that.

Over the next couple months I'll be building Luna, an all-in-one open source Discord bot.

Want to know more? Join the support server!

Features

  • Modmail
  • Tickets
  • Verification channels
  • Levels and Leaderboard
  • Music
  • Custom Commands
  • Profile Pages
  • Real-time Analytics
  • Web Panel
  • Anti-raid
  • Economy
